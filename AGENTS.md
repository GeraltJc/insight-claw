## Agent skills

### Issue tracker

本项目的需求、缺陷和 PRD 使用 GitHub Issues 管理，仓库为 `GeraltJc/insight-claw`。详见 `docs/agents/issue-tracker.md`。

### Triage labels

本项目使用 mattpocock/skills 默认的五类 triage 标签。详见 `docs/agents/triage-labels.md`。

本项目使用 single-context 布局：根目录 `CONTEXT.md` 作为项目上下文，`docs/adr/` 记录架构决策。详见 `docs/agents/domain.md`。
