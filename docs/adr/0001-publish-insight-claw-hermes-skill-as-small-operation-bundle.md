# Publish Insight Claw Hermes Skill as a Small Operation Bundle

Insight Claw Hermes Skill is published as a small operation bundle that teaches Hermes Agent how to download, configure, run, verify, and troubleshoot Insight Claw, rather than packaging the full Insight Claw repository into the skill. This keeps ClawHub distribution small, avoids bundling project code and generated artifacts into the skill surface, and lets agents reuse or clone the canonical repository when execution is needed.
