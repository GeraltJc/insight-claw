# Triage Labels

工程技能使用五类标准 triage 角色。本文档把这些角色映射到本项目 issue tracker 中实际使用的标签字符串。

| mattpocock/skills 中的角色 | 本项目标签 | 含义 |
| -------------------------- | ---------- | ---- |
| `needs-triage` | `needs-triage` | 需要维护者评估 |
| `needs-info` | `needs-info` | 等待报告者补充信息 |
| `ready-for-agent` | `ready-for-agent` | 信息完整，适合 AFK agent 独立处理 |
| `ready-for-human` | `ready-for-human` | 需要人类实现或判断 |
| `wontfix` | `wontfix` | 不会处理 |

当技能提到某个角色时，例如“应用 AFK-ready triage 标签”，应使用上表中对应的“本项目标签”。

如果 GitHub 仓库中的实际标签名称发生变化，直接修改上表右侧即可。
